# NEON WULF: THE TETHERED CITADEL (v3.6)

A high-fidelity, browser-based kinetic arcade engine featuring vector-based physics, 3D geometry projection, and space-time distortion effects.

## 🚀 Features
- **Kinetic Pull Mechanic:** Drag-and-release movement with a dynamic aim-assist guide.
- **Organic Option Chain:** A "Gradius-style" trailing system with elastic spacing and procedural jitter.
- **Space-Time Ripple Grid:** Interactive background grid that reacts to combat and movement disturbances.
- **Citadel Construction:** Progressively build a 3D-projected orbital station by harvesting cargo from hostile sectors.

## 📊 Sprint Analytics
- **Total Development Time:** ~4.5 Hours (Live Iteration)
- **Token Consumption:** ~142,000 Tokens (Cumulative Context)
- **Iterative Cycles:** 12 Major Versions (from 'Option Harvest' to 'Tethered Citadel')

## 🛠 Technical Stack
- **Language:** HTML5 / JavaScript (ES6+)
- **Rendering:** Canvas 2D API with manual 3D-to-2D projection matrices.
- **Physics:** Custom Vector2 library with friction, acceleration, and distance-buffered trail tracking.
